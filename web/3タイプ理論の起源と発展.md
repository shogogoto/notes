### 2. タイプ理論に対する批判
非可述性 = impredicativity: 自身を含む全体への言及によって循環してること、自己言及性
非可述的定義: {非可述性}をもつ定義
- e.g. sup_A: Aを含む上階全体に言及
- e.g. 代入原理: $a=b \iff \forall{P}(P(a) \iff P(b))$ 全ての述語は"=b"自体を含む
- e.g. リシャールのパラドクス

ポアンカレの論理主義批判: 独自の直感主義に基づく
- ラッセルの諸理論が{非可述的定義}の悪循環による欠陥があると指摘
- 論理主義は数学の厳密化には貢献するかもしれないが進歩には貢献しない
- 数学を支える数学的帰納法や無限への直観は論理学に還元されない

ベリーのパラドクス: {タイプの制約}で防げそうにない{非可述的定義}によるパラドクス
- \*: 20文字以内で名付けできない最小の自然数

### 3. タイプ理論の発展と応用
ラッセル以後、数学基礎理論の主流はタイプ理論ではなく公理的集合論へ
ただしタイプ理論は計算機科学や構成主義数学に応用されている

ラムジーによるラッセルへの批判:
- 意味論的パラドクスと論理的パラドクスを分類
  - 数学において重要なのは後者のみと主張
  - 後者に対処するにはタイプの区別だけで十分

単純タイプ理論:
- @1930 チャーチがラムジーのアイデアを体系化
- 述語の自由変数のタイプによってのみ区別
- 束縛変数では区別されない
- 閉じた言明では同じタイプを持つ


型付きλ計算: 計算規則としての側面を抽象化した記号体系
- 関数型プログラミングの理論的基礎
- 命題や項にタイプが付与されている
- 構文規則にタイプの制約を付加
- チャーチは論理的語彙を拡張して数学的基礎を与える論理体系にしようと頑張った
  - @1930 {self}の体系で矛盾が見つかった
  - これを受けてチャーチ諦める
  - 関数的部分のみが切り離されて独自に発展
- λ項の計算は有限ステップで停止し一意な結果を持つ
- チューリング不完全
<-> 型なしλ計算:
- λ項の計算が停止しない場合がある
- チューリング完全
- フレーゲの論理学とパラドックスを生じない

タイプによる制約は人間の言語と思考の合理的だが不自由な側面と不合理だが自由な側面のトレードオフかも
タイプ制限の強弱による人間の思考などは面白い哲学的テーマ

カリー＝ハワード同型: 証明論の体系と型付λ計算の密接な対応関係
- タイプ -> 命題: 命題をデータタイプの同一視
- λ項 -> 証明: 証明をデータタイプから異なるデータタイプを得る手続きと同一視
- λ項の変換 -> 証明の正規化

λ項: 関数の抽象化
