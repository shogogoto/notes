### 1. 論理学的思考へ 意味論と語用論
  論理に従うことで得られる納得感によって、本当に自由になれるから

  非論理への耽溺は思索を職とする人にむしろ多い
    e.g. ポストモダン哲学: 地道な論理を軽視
      at. フランス
      when. 20世紀

  数学では結果よりも、証明の手続きが正しいことが重視される

  論理学: 証明の方法論を体系化する学問
  証明の方法論: 証明テクニックの精緻化や正当化、根拠の吟味

  証明=推論=論証: ある文が真であることから他の文が真であることを導く作業
    真なる前提から間違いなく新成結論を導く

  論理的証明は内容ではなく形式によって正しさが保証される
  仮定: 真であることが確定できていない文

  命題: 文が表現する内容
    命題の本性は論理学者や哲学者の間で合意されていない
      e.g. 文の意味という実態 or 出来事 or イメージ or 規則

  言語研究の３分野:
    構文論: 語句と語句の関係(文法)を扱う
    意味論: 語句とその内容の関係(指示、真偽etc)を扱う
    語用論: 語句と言語使用者との関係を扱う

    言語の規約に関する分野: 構文論と意味論
      <-> 言語の規約では尽くせない事情: 語用論

  言明=主張: 命題の語用論的側面
    e.g. いつどこで誰がどんな意図で言葉を使ったかによる効果

  P1 |意味の外部の語用論的な現象を意味論で捉え直すことは常に可能
  
  メタ言語: 言語について語る言語
    e.g. 構文論について語る文
      e.g. 「恐竜」は活用形がない
    e.g. 意味論について語る文
      e.g. 「恐竜」は動物の種の名前である

  論理学: 人間共通の思考のあり方をどの特定の言語の語彙や文法にも縛られない\
    諸言語共通の最小限のルールで記号化したもの
    いろいろな意味論を持つあらゆる言語に共通する最大公約数的な構造を抽象化する
    どんな言語でもどんな世界でも確実に成り立つ命題の真偽の組み合わせ
  
  推論は知覚、記憶、情念、意欲などの心的作用を関係づけ意味づける
    e.g. 「あいつの仕業か。*とすると*、この前のアレも...もう怒ったぞ！」

### 2. 真と偽 命題の特性を探る

  真理値: 真 or 偽
  命題は真理値を持つ
    <-> 単なる名詞は真理値を持たない

  真理値に関する世界観:
    実在論: 人間の観測や認識とは独立に世界のあり方は決まっている
      {parent}が最も常識に近い
      <-> 観念論:
    実証主義: 観測されたことだけが確定していて、それ以外は真偽を問えない

  但書: 命題の真偽を左右する語用論的要因(誰にとってどういう文脈なのか)
    e.g. 「日本の首都は東京である」はどの時代の人にとってかで真偽変わる
  {但書}を全て書き尽くすことは原理的に不可能
    -> 命題の真偽を厳密に確定できない
    <-> 寛容の原則: 偽とはっきり言えない命題はなるべく真となるように解釈する
      e.g. 「今日はいい天気である」は雨とか一面雲ではない限り厳密にはよくわからんけど真


  本当の真理とは何か、ではなく、真理という概念がどうのような構造をもつか

  論理演算子: 文と文を結びつけて新たに文を作る接続詞
    否定: 「でない」 $\neg$
    選言: 「または」 $\lor$
    連言: 「かつ」   $\land$
    含意: 「ならば」 $\supset$

  真理表: 論理演算子の入出力の真理値の対応表
    {寛容の原則}で判断すると分かりやすい

  二値論理学: 真理値は真と偽の２通りだけ
    <-> 多値論理学:

  命題型: 不特定の具体的命題を表しうる表現
    e.g. $P, Q, P \lor Q, P \supset Q$

  真理関数: 真理値の入出力をもつ関数


### 3. トートロジー 偽となりえない命題

  １つの論理演算子に対応する言い回しは複数ある
    語用論の問題
    e.g. かつ、しかし、のみならず、にもかかわらず、しかも、なのに
    e.g. または、か、さもなくば、でなければ

  論理学では排他的選言はほとんど使われない
    <- 連言と相補的でない
    <- 両方真の場合に偽になるなんて{寛容の原則}に反する

  原子命題: これ以上分解できない基本的な命題
  分子命題: 原子命題を論理演算子で結合した命題
  主演算子: 分子命題で最後に適用された演算子
  トートロジー=恒真式=論理的真理: いかなる真理値の組み合わせでも真となる分子命題
    いかなる可能世界においても真
    <-> 恒偽式=矛盾式:
    e.g. 同語反復: $P \supset P${トートロジー}の語源
    e.g. 排中律: $P \lor \neg P$
    e.g. 矛盾律: $\neg(P \land \neg P)$

  {トートロジー}は意味論的な価値はないが、語用論的には意味がありうる

### 4.「ならば」の論理 条件文の構造
  前件: {含意}の前にくる文
    <-> 後件:

  {含意}は論理学で最も中心的な演算子

  取引解釈: 契約や警告、脅迫に用いられる「ならば」の自然な解釈
    e.g. 「入会すれば、施設を無料で使えます」と言われて入会したのに施設が有料だったら嘘だと思う
    {前件}が偽,{後件}が真の{含意}を偽に対応付ける

  {前件}が偽のとき{後件}の真偽に依らずに含意は真
    <- {前件}が偽のとき、もはや{含意}の文の真偽は曖昧だから真とみなす
      <- {寛容の原則}

  含意の真理表の直感的説明:
    「PならばQ」と「PなのにQでない、ということはない」の真理表が一致する
    「PまたはQ」は「PでないならばQ」
    e.g. P41|「XXXならば明日ヒョウが降るよw」
      {前件}が偽と主張するレトリック
      有り得なそうな{後件}を言っても{前件}が偽ならば嘘ついたことにならない言いたい
  
  {前件}と{後件}は関連していなくてもよい
    e.g. `P41`


  十分条件: 真の{含意}の{前件}
    {十分条件}が真であるならば{後件}も真であると確定する
      -> {十分条件}が真であるさえ分かれば{後件}の真偽を知るのに十分である
    <-> 必要条件: 真の含意の後件
      {前件}が真であるためには少なくとも{必要条件}が真である必要がある
        <- {必要条件}が真であっても{前件}は真偽の２通りがある
        <- {必要条件}が偽ならば{前件}も偽であると確定する

  必要十分条件=双条件=同値: {必要条件}かつ{十分条件}
    {前件}と{後件}の可能的状況の範囲が一致する
      -> {前件}と{後件}の{真理値}が等しい

  強い命題: {トートロジー}な含意における{十分条件}
    <-> 弱い命題: {トートロジー}な含意における{必要条件}

  いかなる可能世界でも{強い命題}は{弱い命題}の真偽を制約する
    <-> 単なる{必要条件}や{十分条件}では可能世界によっては包含関係が成り立たない

  {トートロジー}は最も弱い命題である
    <- {トートロジー}を{後件}においても{十分条件}の真偽については何もいえない
    いかなる可能性回でも真となってしまうほど弱いと解釈できる
  {恒偽式}は最も強い命題である
    <- {恒偽式}を{前件}におくとその含意は{後件}に依らず、いかなる可能世界でも真
    いかなる可能世界でも真となれないほど強いと解釈できる

  ド・モルガンの法則:
    1. $\neg(P \lor Q) \equiv (\neg P \land \neg Q)$
    2. $\neg(P \land Q) \equiv (\neg P \lor \neg Q)$

  非真理関数: 入力の{真理値}だけでは出力の{真理値}は決まらない
    e.g. 可能である、許されている、義務である、と知られている、望ましい

### 5. 「妥当な推論」とは何か 推論規則と定理
  妥当: 前提が真であれば結論が真であることが保証されること

  公理: {妥当}な推論を最初に適用する出発点
    論理というゲームのための便宜的出発点
  定理: {公理}から導かれる真なる命題

  {公理}の真偽は公理系によって変わる
    {公理}は議論ごとに設定してよい
      e.g. 「歩」の動きは将棋と詰将棋で異なる
    {公理}が決まれば{定理}も決まる

  妥当な推論規則: {含意}を{主演算子}とするトートロジーへの対応をもつ前提と結論の組
    e.g. 否定除去規則:
      前提 $\neg\neg{P}$
      結論 $P$
    e.g. 前件肯定規則
      前提
        $P \supset Q$
        $P$
      結論 $P$
    e.g. 対偶法
      前提 $P \supset Q$
      結論 $\neg Q \supset \neg P$
    e.g. 選言除去規則
      前提
        $P \lor Q$
        $P \supset R$
        $Q \supset R$
      結論 $R$
    e.g. 選言三段論法
      前提
        $P \lor Q$
        $\neg P$
      結論 $Q$
    e.g. 仮言三段論法 ~ 推移律
      前提
        $P \supset Q$
        $Q \supset R$
      結論 $P \supset R$
    e.g. 移出法
      前提 $P \supset (Q \supset R)$
      結論 $(P \land Q) \supset R$
    e.g. 連言導入規則
      前提 
        $P$
        $Q$
      結論 $P \land Q$

  {トートロジー}はあらゆる公理系の定理
    -> 数学だろうが物理学、心理学だろうが{トートロジー}は否定できない
    -> あらゆる可能世界で通用する{トートロジー}な推論規則を整備するのが論理学
  健全: {妥当}かつ前提が全て真である
  
  {妥当}な推論は前提に含まれてる以上の情報をもたらさない

### 6. 推論の冒険 演繹定理から仮説演繹法へ


  演繹定理: 前提に{仮定}を含めたらある結論Qが得られる時、\
    PならばQという条件付きの弱めた結論を導く
    {妥当な推論規則}にトートロジーな式を対応させるメタ定理
  同値式導入規則: {主演算子}が{同値}である{トートロジー}の左辺と右辺は任意の命題で置換可能

  補助前提: 暗黙の隠れた諸前提
  後件肯定の誤謬: $P \supset Q$と$Q$から$P$を結論する妥当でない推論

  仮説演繹法: $仮説 \supset 予想$を{演繹定理}から得て{後件肯定の誤謬}によって仮説を立証する
    誤謬なのにうまくいく理由
      1. 仮説なしには到底起こり得ないような驚くべき予想$\neg P \supset \neg Q \equiv Q \supset P$\
        であり、予想が正しければほとんど{前件肯定}が成り立つ
      2. いろんな予想を繰り返し確認することで{十分条件}である仮説の確度を高める

  演繹: {妥当}な推論
    <-> 帰納: {妥当}ではない場合があるが前提には含まれない情報を含む結論を求める攻めの推論

  {仮説演繹法}にて予想が外れた場合、{仮説}は{演繹}的に反証される
    <- 後件否定規則$((P \supset Q) \land \neg Q) \supset \neg P$

    1. 前提が事実に反すると指摘する
    2. 名目的再定義: 前提が真となるような定義へ変更する

        弱い主張への置き換えは{演繹定理}と似ている

  仮説演繹法による反証は{背理法}に書き直せる

### 8. 述語論理学 命題の内部構造を探る

### 9. 多重量化と同一性 日常言語の曖昧さを解きほぐす

  量化子の順序によって意味が変わる
    1. $\forall{x}\exists{y}R(x, y=t)$
