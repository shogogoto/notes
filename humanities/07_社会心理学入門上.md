## 1. ゲシュタルト心理学
経験説: そう知ってるからそう知覚する
  <- 盲人が手術で見えるようになった直後でも月とピンポン玉を見比べたときに月は遠くに見えた
ゲシュタルト心理学: 知覚はその場の中の一部として全体との関係の中で理解される
  when. 19C EARLY
  <- 脱衣は風呂場では普通、外では変態、というように場で意味付けが左右される
  <- 要素心理学: 心理は単なる構成要素の集まりに還元されるという{ゲシュタルト心理学}以前の立場
  -> 対比, contrast: 同一場の2要素の知覚の差異が大きく見える
  -> 同化, assimilation: 同一場の2要素の知覚の差異が小さく見える
  -> 最大秩序の法則: 諸要因に基づいて最も簡単で秩序立ったものが場から離れて理解される
    近接, 共同運命, 類似, 共通教会, よい形態, 閉鎖
帰因, 帰属: 行動の原因であり、なぜそれが起きたかを説明づける
常識心理学: 自他の行動の解釈を与えて社会生活を可能とするもの個々人の素朴な心理学
  -> `印象`
単一要因の{帰属}: 大衆受けする単純明快な１つの断定された{帰属}
行為者は{帰因}を自身の特性に求めるのを避ける
観察者は{帰因}を行為者の特性だと思いがち
成就要求, 達成動機: 優れた成果を出して自尊心を高めたい
フリッツ・ハイダー: 個人の行動を理解するためにはその人の{常識心理学}を理解する必要が有る
  when. 1896 ~
  -> 成功失敗の{帰因}は能力 努力 困難度 運 の４つ
    -> 高{成就要求}な人は成功時は能力、失敗時は努力や運のせいにする
      めげずに努力し続ける
  -> 均衡理論: 3者間の正負の関係を掛けて正なら安定、負なら不安定
    ex. 私の好きな人(+)が私の嫌いな人(-)と仲がいい(+)と気持ち悪い
    正負でない中立も、強度も扱えない
ケリーの特性: 個人固有の{帰因}として判断する4要因
  弁別性: 特定の印象をその人以外では感じられない
  一貫性: 特定の人に対していつも類似の印象を抱く
  様態を超えた{一貫性}: あらゆる関わり方においても抱く{一貫性}
  観察者の意見の一致: 皆同じ印象を持っている
印象: 他人の行動を予想するための仮説
  絶えず誰もがやってる
  無意識や文化や価値観が{印象}の評価に影響する
中核的: 1つ人の特性といろんな他の特性を同時に持つ度合い
潜在人格理論: {中核的}な特性から人物像を推論し{印象}を形成し、その人の行動を予測しようとする
  文化によって異なる
    <- 素直は非を認めたり(upright)傾聴しそれに従う(compliant)が{中核的}だが、米国では全く別の特性
ソロモン・アッシュ, アッシュ: 自己は社会との関わりの中で形成されると考えていたユニークな社会学心理学者
  when. 1907 ~
  -> 温かい人 or 冷たい人 と紹介された人の20min講義 学生のその人の{印象}を調査
    温かい人では発言が56% 冷たい人では32%へ低下
良い、悪いが最も{中核的}
  when. 1916
  <- チャールズ・オズグッド: SD法を発明し研究領域を拓いたイェール大
初頭効果: 最初の{印象}が全体の{印象}を決める
  多くの情報を一度に与えると{初頭効果}が強くなる
親近効果: 最後の{印象}が全体の{印象}を決める
  小出しで漸次{印象}形成すると{親近効果}が強くなる
{アッシュ}の実験: このテープと同じ長さのものは1,2,3番のどれ？と5人(前から4人サクラ)に順に尋ねる
  when. 1950
  <- `アッシュ`
  アメリカで男子学生のみ
  明らかな誤りを答えたサクラに同調せずに正しく回答したのはわずか 13/50
    <- 中枢神経系が以上興奮すると血中の脂肪酸濃度が上がる
    <- サクラに気づかず(X-G+ A-G+)皆と同じ回答に曲げる(X-A-to+)
      被験者X 回答者たちG 皆の回答A
      X-G = サクラに気づかず同胞と思う(+), 他の奴は馬鹿(-)
      X-A = 自分の回答と皆のが一致(+) or not(-)
      A-G = 皆素直に答えてる(+) or サクラが嘘ついてる(-)
      <- `均衡理論`
  サクラは1人でも効果あり、↗2↗3→4  3人まで効果増幅, 4人で果変わらず
  -> フレイザーの追試: {アッシェの実験} at 慶応大 非同調率 34/128
    when. 1966
  -> 佐古の追試: {アッシェの実験} at 阪大 非同調率 11/22
    when. 1975
自動運動現象: 暗闇の小さな灯が止まっていても動いてるように見える
シェリフの実験: {自動運動現象}が3人集まると各自の見え方の癖に反して同じ見方をするようになる
同調しやすい人は劣等感や不完全感を持つ傾向
日本人は同調的と思われていたのに個人主義のアメリカ人非同調な行動を取った
  <- `アッシュの実験`
  <- `フレイザーの追試`
  <- `佐古の追試`
  <- 欧米人は各々が好きに注文するが日本人は皆生ビールを注文したりする



## 2. 場の理論
心理学的環境: 行動に影響を与える環境の部分
非{心理学的環境}: {心理学的環境}以外の物理的環境とか全部
生活空間: 個人 + {心理学的環境}
  どの{心理学的事実}が含まれるかは時々刻々と変わり得る
心理学的事実: {生活空間}の要素で行動に影響するモノの状態や条件
浸透性: {生活空間}に
事象: {心理学的事実}の相互作用

細胞: 個人内の要素
知覚・運動領域, P-M: 個人と外界と接する境界
内的・個人領域, I-P: {知覚・運動領域}のに包まれた内部
周辺的{細胞}, P: {P-M}に隣接する個人の要素
中心的{細胞}, C: {I-P}のように中心にある要素
材質: {心理学的事実}や{細胞}の性質 遠近、強弱、流動硬直など
  流動性
    疲労や興奮で高まる


緊張, tension: 要求needが生じてそれを満たしたい状態
誘発性, valence: {緊張}を軽減してくれるものとしての誘引 + or -
力, force: {誘発性}により{心理的事実}を引き付けたり遠ざけたりする力
均衡状態: {緊張}のレベルが揃っている
  -> `均衡理論`
拡散; {均衡状態}を目指して{緊張}が{移動}する
移動, locomotion: {緊張}により
  空想や夢は{移動}の一種

{心理学的事実}を跨いである行動へ至る
  {力}に導かれて行動

代償行動: {緊張}を減らすための行動の代わり

ザイガルニーク指数: 想起された未完了作業数 / 想起された完了作業数
  やりかけの作業が気になってしまうという当たり前を実証
  被験者に自由時間を与えたら未完了作業を再開した 82%
    {成就要求}が強い人の再開率高い
  自由時間を与えるのを遅らせると未完了作業の再開率下がった
    <- `拡散`

未完了作業は{緊張}を残す
  -> ザイガルニーク現象: {ザイガルニーク指数} > 1
    実験では1.9くらい
  -> 時間が立つと{ザイガルニーク指数}は1へ近づいた
    <- `拡散`

完了せずにプライドが傷つくものは想起されにくい
  -> {ザイガルニーク指数} < 1
