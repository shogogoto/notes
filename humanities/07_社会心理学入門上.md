## 1. ゲシュタルト心理学
経験説: そう知ってるからそう知覚する
  <- 盲人が手術で見えるようになった直後でも月とピンポン玉を見比べたときに月は遠くに見えた
ゲシュタルト心理学: 知覚はその場の中の一部として全体との関係の中で理解される
  when. 19C EARLY
  <- 脱衣は風呂場では普通、外では変態、というように場で意味付けが左右される
  <- 要素心理学: 心理は単なる構成要素の集まりに還元されるという{ゲシュタルト心理学}以前の立場
  -> 対比, contrast: 同一場の2要素の知覚の差異が大きく見える
  -> 同化, assimilation: 同一場の2要素の知覚の差異が小さく見える
  -> 最大秩序の法則: 諸要因に基づいて最も簡単で秩序立ったものが場から離れて理解される
    近接, 共同運命, 類似, 共通教会, よい形態, 閉鎖
帰因, 帰属: 行動の原因であり、なぜそれが起きたかを説明づける
常識心理学: 自他の行動の解釈を与えて社会生活を可能とするもの個々人の素朴な心理学
  -> `印象`
単一要因の{帰属}: 大衆受けする単純明快な１つの断定された{帰属}
行為者は{帰因}を自身の特性に求めるのを避ける
観察者は{帰因}を行為者の特性だと思いがち
成就要求, 達成動機: 優れた成果を出して自尊心を高めたい
フリッツ・ハイダー: 個人の行動を理解するためにはその人の{常識心理学}を理解する必要が有る
  when. 1896 ~
  -> 成功失敗の{帰因}は能力 努力 困難度 運 の４つ
    -> 高{成就要求}な人は成功時は能力、失敗時は努力や運のせいにする
      めげずに努力し続ける
  -> 均衡理論: 3者間の正負の関係を掛けて正なら安定、負なら不安定
    <- `一貫性の原理`
    ex. 私の好きな人(+)が私の嫌いな人(-)と仲がいい(+)と気持ち悪い
    正負でない中立も、強度も扱えない
ケリーの特性: 個人固有の{帰因}として判断する4要因
  弁別性: 特定の印象をその人以外では感じられない
  一貫性: 特定の人に対していつも類似の印象を抱く
  様態を超えた{一貫性}: あらゆる関わり方においても抱く{一貫性}
  観察者の意見の一致: 皆同じ印象を持っている
印象: 他人の行動を予想するための仮説
  絶えず誰もがやってる
  無意識や文化や価値観が{印象}の評価に影響する
中核的: 1つ人の特性といろんな他の特性を同時に持つ度合い
潜在人格理論: {中核的}な特性から人物像を推論し{印象}を形成し、その人の行動を予測しようとする
  文化によって異なる
    <- 素直は非を認めたり(upright)傾聴しそれに従う(compliant)が{中核的}だが、米国では全く別の特性
ソロモン・アッシュ, アッシュ: 自己は社会との関わりの中で形成されると考えていたユニークな社会学心理学者
  when. 1907 ~
  -> 温かい人 or 冷たい人 と紹介された人の20min講義 学生のその人の{印象}を調査
    温かい人では発言が56% 冷たい人では32%へ低下

  <- チャールズ・オズグッド: SD法を発明し研究領域を拓いたイェール大
初頭効果: 最初の{印象}が全体の{印象}を決める
  多くの情報を一度に与えると{初頭効果}が強くなる
親近効果: 最後の{印象}が全体の{印象}を決める
  小出しで漸次{印象}形成すると{親近効果}が強くなる
{アッシュ}の実験: このテープと同じ長さのものは1,2,3番のどれ？と5人(前から4人サクラ)に順に尋ねる

## 2. 場の理論
トポロジー: 大きさや量や形を無視して空間の全体と部分の関係を扱う数学分野
ホドロジー: 目的に達する経路を研究する数学分野
{トポロジー}心理学, {ホドロジー}心理学, ベクトル心理学: {生活空間}のゾウリムシみたいな絵で行動を説明する
  ただ行動の結果を図示しただけで行動予測に使えないという批判
  <- クルト・レヴィン: 実験や更新育成で最も多きな影響を残した心理学者の一人
    when. 1890 ~ 1947
  心理学的環境: 行動に影響を与える環境の部分
  非{心理学的環境}: {心理学的環境}以外の物理的環境とか全部
  生活空間: 個人 + {心理学的環境}
    どの{心理学的事実}が含まれるかは時々刻々と変わり得る
  心理学的事実: {生活空間}の要素で行動に影響するモノの状態や条件
  事象: {心理学的事実}の相互作用
  細胞: 個人内の要素
  知覚・運動領域, P-M: 個人と外界と接する境界
  内的・個人領域, I-P: {知覚・運動領域}のに包まれた内部
  周辺的{細胞}, P: {P-M}に隣接する個人の要素
  中心的{細胞}, C: {I-P}のように中心にある要素
  材質: {心理学的事実}の影響の受けやすさ
    流動性: {移動}によって行動として選択されやすい 疲労や興奮で高まる
    浸透性: {移動}が通過しやすい

  緊張, tension: 要求needが生じてそれを満たしたい状態, 心的エネルギーが高まった状態
  誘発性, valence: {緊張}を軽減してくれるものとしての誘引 + or -
  力, force: {誘発性}により{心理的事実}を引き付けたり遠ざけたりする力
  均衡状態: {緊張}のレベルが揃っている
    -> `均衡理論`
  拡散; {均衡状態}を目指して{緊張}が{移動}する
  移動, locomotion: {緊張}が伝わって経路が繋がる
    空想や夢は{移動}の一種
  拡散: {緊張}のエネルギーが広がっていく
    -> 貧乏ゆすりや怒りが爆発
  {心理学的事実}を跨いである行動へ至る
    {力}に導かれて行動

ザイガルニーク指数: 想起された未完了作業数 / 想起された完了作業数
  やりかけの作業が気になってしまうという当たり前を実証
  被験者に自由時間を与えたら未完了作業を再開した 82%
    {成就要求}が強い人の再開率高い
  自由時間を与えるのを遅らせると未完了作業の再開率下がった
    <- `拡散`
未完了作業は{緊張}を残す
  -> ザイガルニーク現象: {ザイガルニーク指数} > 1
    実験では1.9くらい
  -> 時間が立つと{ザイガルニーク指数}は1へ近づいた
    <- `拡散`
完了せずにプライドが傷つくものは想起されにくい
  -> {ザイガルニーク指数} < 1
代償行動: 本来のとは異なる{流動的}な行動によって{緊張}を減らす
  似た行動は{代償行動}になりやすい
  ポルノ解禁で覗きや児童へのいたずらは目立って減少したが強姦などは変わらず
  暴力映像が暴力を促進するかの一定した実験結果はない
心的飽和, 過剰飽和: 同じことに飽きてイライラ、疲労、肩こり
  {緊張}が下がっていく
  作業単位が大きく単調 or それが当人に重要である場合に起こりやすい
共飽和: 似た作業にも{心的飽和}が広がる
要求水準: 個人の目標の達成難易度
  前回の成功 * 次回の成功確率 - 前回の失敗 * 次回の失敗確率 で更新される
  <- `成就要求`
成功: {要求水準}以上
失敗: {要求水準}未満


協和, 均衡: {均衡理論}で正、{緊張}がない
認知的不{協和}: Aかつnot A による矛盾した不快な状態で、それを解消するように駆り立てる
  <- `一貫性の原理`
  不{協和}を解消する行動を必ず取るはず、この実験で不{協和}な状態になったに違いない、という強い決めつけ
  行動予測には曖昧すぎる
  <- レオン・フェスティンガー: {レヴィン}の弟子
    when. 1919 ~
    -> アンケートの平均値 A, B, C = -4.5, +1.33, -0.05
      超退屈な作業をさせて面白いとサクラから聞かされたら反応が変わるのか、という実験だと伝えた
        サクラはいないし嘘の説明

        当時30セントで山盛りバーガー食えたので$20は割と大金
      <- Aは素直な反応、Cは大金をもらってる(+)の詰まらないと肯定できる(-)、Bはできない
        <- `認知的不協和`
    ->  キーチ夫人: lake cityが大洪水になるがクラリオン星人の信者は宇宙船で救うと自動書記で予言した夫人
      大洪水に備えて仕事をやめたり夫人宅家に集まった人ほど{認知的不協和}が強く、予言外れた後の方が信心深くなった
        大洪水は信者の信仰心の結果起こらなかったという説明
      夫人宅に集まらなかったライトな信者は予言外れた後に信者やめた
  -> 必修でない授業が詰まらないのは困るから、授業が面白かったと思うことにする
  -> ボーナスで欧州旅行、でも最悪だった。何か良かった点を必死に探す

  自由に話せない者は被験者に向いてない、参加してもよい？と念押し(A)

適合性理論: 愛憎(好き嫌い)を-3..+3で表し、同じ数値同士の項目が+で or 正負が-で繋がると適合
  <- 一貫性の原理: 大切な事柄には皆の賛同を得たいし、嫌悪する事柄には皆に非難してほしい
  -> 態度変容: +関係の不適合は2点間の内分点へ移って適合 1, 3 -> 2.5 3-1=2を3:1に内分
    -関係の不適合は正負同じ位置にあった場合の内分点にそれぞれ移る
    0にあったものはもう一方の要素の位置に移動して適合状態になる
  不信任による修正: 正同士が-関係 or 負同士が+関係の時にそんな馬鹿な、と{態度変容}が起きない
  主張定数: 関係の方向、好いた者よりも好かれた者の方が余分に評価される分
    {適合性理論}のズレを修正する試みだが不完全
価値連合の原則, 威光暗示: 高い価値のあるものと結びつけることでその価値が高まる
